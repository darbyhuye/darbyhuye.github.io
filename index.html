Publications

SOK: Identifying Mismatches Between Microservice Testbeds and Industrial Perceptions of Microservices (Seshagiri and Huye et al., JSys'22) Microservices are widely used to build distributed systems in industry. Academics are left in the dark when doing research in this space as we do not have access to these industrial microservice applications. Many researchers have attempted to make test applications for academic use, but these applications are designed using narrow design decisions that are not necessarily representative of the design decisions made in their industrial counterparts. We interviewed folks in industry about their experiences with microservices and compared these applications with the academic testbeds.

Automating instrumentation choices for performance problems in distributed applications with VAIF (Toslali et al., SOCC '21) If we lived in a world where every line of code in a system had instrumentation describing it, we would have complete visibility into the system that would allow us to easily identify sources of performance issues. However, this is infeasble due to information overload (on the system and the human interpreting it).
